\section{Laserschneiden}
	Beim Laserschneiden können meist flache Werkstoffe verschiedenster Dicke mithilfe von einem konzentrierten Laserstrahl ohne Berührung geschnitten werden. Der Laserstrahl erhitzt dabei das Material an einem Punkt so stark, dass es sofort schmilzt oder verdampft. Mithilfe von einem Schneidgas kann das nun geschmolzene Material aus der Schnittfuge geblasen werden. Mit dieser Technik können eine Vielzahl an Materialien, wie zum Beispiel Holz, Acryl, Schaumstoff oder auch diverse Metalle wie Stahl, Edelstahl und Aluminium. \cite{Trumpf2024}
