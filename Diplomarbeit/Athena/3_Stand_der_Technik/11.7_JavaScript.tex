JavaScript ist eine Programmiersprache, die für die Entwicklung interaktiver Webseiten verwendet wird. Mit JavaScript können Entwickler Benutzerinteraktionen wie Klicks, Eingaben und Mausbewegungen verarbeiten, Inhalte dynamisch aktualisieren und mit dem Server kommunizieren, um Daten auszutauschen. JavaScript wird direkt in HTML-Dokumente eingebettet und von Webbrowsern ausgeführt, was es zu einer der wichtigsten Technologien für die Webentwicklung macht. \cite{seokueche.de2024a}
	\subsubsection{Grundlagen von JavaScript} 
Syntax und Struktur von JavaScript-Code: 
JavaScript-Code wird in kleine Abschnitte unterteilt, die als Anweisungen bezeichnet werden. Diese Anweisungen folgen einer bestimmten Struktur, die der Computer versteht. 

Datentypen, Variablen und Operatoren: 
In JavaScript gibt es verschiedene Arten von Informationen, die als Datentypen bezeichnet werden, wie zum Beispiel Zahlen und Wörter. Variablen sind wie Behälter, in denen diese Informationen gespeichert werden können. Operatoren sind wie Werkzeuge, um mit diesen Informationen zu arbeiten, zum Beispiel um sie zu addieren oder zu vergleichen. 

Kontrollstrukturen wie Bedingungen und Schleifen: 
Mit JavaScript können wir Regeln erstellen, die bestimmen, was passiert, wenn bestimmte Bedingungen erfüllt sind. Zum Beispiel können wir sagen: "Wenn es regnet, dann nimm einen Regenschirm." Schleifen sind wie Wiederholungen von Anweisungen. Zum Beispiel könnten wir sagen: "Wiederhole diese Anweisung fünfmal." \cite{seokueche.de2024a}
		\subsubsection{DOM-Manipulation} 
DOM bedeutet Document Object Model 
Interaktion mit HTML-Elementen: 
Die Document Object Model (DOM)-Manipulation ermöglicht es JavaScript, auf HTML-Elemente zuzugreifen und sie zu ändern. Durch die Verwendung von JavaScript können Entwickler Elemente finden, erstellen, ändern oder entfernen und auf Ereignisse wie Klicks oder Eingaben reagieren. 
			\subsubsection{Dynamische Manipulation von Inhalten, Stilen und Ereignissen} 
Inhalte: JavaScript kann verwendet werden, um den Textinhalt von HTML-Elementen zu ändern, neue Elemente hinzuzufügen oder bestehende Elemente zu entfernen. 

Stile: Mit JavaScript können Entwickler die CSS-Eigenschaften von HTML-Elementen ändern, um ihr Aussehen dynamisch anzupassen. 

Ereignisse: Entwickler können JavaScript verwenden, um auf Benutzerinteraktionen wie Klicks, Mausbewegungen oder Tastatureingaben zu reagieren und entsprechende Aktionen auszuführen. 

Durch die DOM-Manipulation können Webseiten besser gestaltet und Benutzaktionen angepasst werden, was zu einer besseren Benutzererfahrung führt. 
Ereignisbehandlung 
Ereignisbehandlung ist die Implementierung von Funktionen, die auf Benutzerinteraktionen wie Klicks, Mausbewegungen oder Tastatureingaben reagieren. Diese Funktionen werden dann mit den entsprechenden HTML-Elementen verbunden, um bestimmte Aktionen auszuführen, wenn das Ereignis eintritt.\cite{seokueche.de2024a}
				\subsubsection{AJAX und Datenübertragung} 
AJAX ermöglicht es, Daten zwischen dem Browser und dem Server auszutauschen, ohne die Seite neu zu laden. Diese Technik erlaubt es, Teile einer Webseite dynamisch zu aktualisieren. Daten können im JSON- oder XML-Format zwischen Client und Server übertragen werden. Typische Anwendungsfälle sind das Laden von Inhalten, das Überprüfen von Formularen oder die Verarbeitung von Benutzerinteraktionen. AJAX-Anfragen werden mit JavaScript-Funktionen wie XMLHttp Request oder fetch() durchgeführt.\cite{seokueche.de2024a}
					\subsubsection{Überblick über gängige Bibliotheken und Frameworks} 
jQuery: Vereinfacht die Arbeit mit JavaScript, insbesondere die Manipulation von DOM-Elementen und die Ausführung von AJAX-Anfragen. 

React: Eine Bibliothek für den Aufbau von Benutzeroberflächen, die komponentenbasierte UIs für Webanwendungen ermöglicht. 

Angular: Ein Framework für die Entwicklung von umfassenden Webanwendungen mit Werkzeugen für Datenmanagement, Routing und Komponentenerstellung. 

Vue.js: Ein Framework für die Entwicklung von flexiblen Benutzeroberflächen. 

Bootstrap: Eine Bibliothek für das responsive Design von Webseiten mit vorgefertigten Komponenten und Layouts. 

Express.js: Ein Framework für die Entwicklung von serverseitigen Anwendungen mit Node.js, besonders gut für die Erstellung von APIs geeignet.\cite{seokueche.de2024a}

D3.js: Eine Bibliothek für die Datenvisualisierung mit vielfältigen Möglichkeiten zur Erstellung von Diagrammen und interaktiven Visualisierungen.
