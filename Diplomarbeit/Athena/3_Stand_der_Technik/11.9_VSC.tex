\subsection{Visual Studio Code}

			\centering
			\includegraphics[scale=0.45]{./3_Stand_der_Technik/Abbildungen/VSC_LOGO}

Visual Studio Code ist ein Code Editor der kostenfrei ist. Er wurde von Microsoft entwickelt. Gründe warum VSC so beliebt ist, sind zum Beispiel die Benutzerfreundliche Oberfläche und eine umfangreiche Erweiterbarkeit. Hier sind einige Hauptmerkmale beschrieben:\cite{biteno.com2024}
\begin{itemize}
\item Benutzerfreundliche Oberfläche: Visual Studio Code bietet eine übersichtliche und anpassbare Benutzeroberfläche, die eine effiziente Bearbeitung von Code ermöglicht.
Integrierte Entwicklungsumgebung (IDE): Obwohl Visual Studio Code leichtgewichtig ist, bietet es dennoch viele Funktionen einer vollwertigen IDE, einschließlich Syntaxhervorhebung, Codevervollständigung, Debugging-Tools und integrierter Git-Unterstützung.\cite{biteno.com2024}

\item Unterstützung für verschiedene Programmiersprachen: Visual Studio Code bietet Unterstützung für eine Vielzahl von Programmiersprachen und Technologien, darunter HTML, CSS, JavaScript, Python, C++, Java und viele mehr.
Erweiterbarkeit: Eine der herausragenden Eigenschaften von Visual Studio Code ist seine Erweiterbarkeit. Durch die Verwendung von Erweiterungen können Entwickler Funktionalitäten und Tools hinzufügen, um ihre Arbeitsabläufe zu verbessern und an ihre Bedürfnisse anzupassen.\cite{biteno.com2024}

\item Versionskontrolle: Visual Studio Code bietet eine nahtlose Integration mit Git und anderen Versionskontrollsystemen, was die Verwaltung von Codeprojekten und die Zusammenarbeit im Team erleichtert.
Plattformübergreifend: Visual Studio Code ist für Windows, macOS und Linux verfügbar, was es zu einer vielseitigen Wahl für Entwickler macht, die auf verschiedenen Betriebssystemen arbeiten.
Insgesamt ist Visual Studio Code ein leistungsstarker und flexibler Code-Editor, der sich sowohl für Anfänger als auch für erfahrene Entwickler eignet und eine effiziente Entwicklungsumgebung für eine Vielzahl von Programmiersprachen und Technologien bietet.\cite{biteno.com2024}
\end{itemize}
